Magnus Sidekick - Browser Extension
A simple bridge to connect your browser to Magnus Cyberdeck.

What it does:
‚Ä¢ Sends active tab content to Magnus (automatically added as context in chat)
‚Ä¢ Real-time page monitoring
‚Ä¢ Seamless integration with Cyberdeck chat

Installation (5 Steps):
1. Extract the sidekick zip to your home directory (e.g., ~/magnus-sidekick/)
2. Open Chrome/Firefox/Edge and enable Developer Mode
3. Click [Load Unpacked]
4. Navigate to ~/magnus-sidekick/extension and open it
5. Pin the Sidekick icon for easy access

How to Use:
1. Navigate to any webpage you want to analyze
2. Click the Sidekick icon to toggle ON (icon lights up)
3. Check Cyberdeck toolbar - you'll see the page referenced
4. Chat with Maggie about the page content
5. Toggle OFF to stop streaming
